# Немного практики алгоритмов

## Описание
BFS и DFS на python, c++

## Технологии
### Для Python
  * heapq
### Для С++
  * queue
  * vector
  * stack
  * unordered_set

## Как запустить

1. Склонируйте репозиторий
```
git clone <URL>
cd <name>
```
2. Создайте и активируйте виртаульное окружение (`python`)
```commandline
python -m venv venv
source venv/Scripts/activate
```
